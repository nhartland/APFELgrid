# APFELgrid
Code and examples for the APFELgrid plugin.

Requirements
------------
**APFEL** master

**libnnpdf** master

**APPLgrid** 1.4.70

How to run the example
----------------------
For the example to run correctly, first you need to download
the required APPLgrid files, by running the script **/grids/fetchTest.sh**.

Once complete, enter

*make*

*./example*
